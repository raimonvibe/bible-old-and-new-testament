# General epistles capitalization pass (Hebrews – Jude)

Applied 2026-07-22 via `scripts/apply-general-epistles-capitalization.js` (~55 replacements) plus manual fixes for Hebrews 9–11 and 1 John.

## Books covered

Hebrews, James, 1–2 Peter, 1–3 John, Jude.

## Notable fixes

| Book / chapter | Fix |
|----------------|-----|
| **Heb 1** | **His Son**, **He** appointed, **He** made worlds, Psalm quotes |
| **Heb 2–4** | **He** subjected all things; **His** rest and voice |
| **Heb 5–10** | Christ priesthood — **He/Him/His** in incarnation and sacrifice |
| **Heb 7** | Melchizedek blessed **him** (Abraham, human) |
| **Heb 11** | Abel/Enoch/Moses → **he/him** (human); **He** exists (God) |
| **Heb 12** | Consider **Him** who endured — ** Himself** |
| **James 4** | World friendship → **himself**; devil → **he** will flee |
| **1 Pet 1–3** | Love **Him** unseen; **He** who called; Christ suffered → **He** |
| **2 Pet 1** | **His** divine power; **Him** who called |
| **1 John 1–5** | **He** is faithful; know **Him**; prayer to **Him** |
| **Jude 1** | **He** has kept angels; **Him** who is able — **His** glory |
