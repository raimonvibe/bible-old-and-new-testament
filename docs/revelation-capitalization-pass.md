# Revelation capitalization pass

Applied 2026-07-22 via `scripts/apply-revelation-capitalization.js` (~37 replacements).

## Scope

All 22 chapters — Christ, God on the throne, and the Lamb; lowercase for overcomers, beasts, dragon, and angels.

## Notable fixes

| Chapter | Fix |
|---------|-----|
| **1** | **He** is coming; vision of Christ — **His/Him/He** throughout |
| **2–3** | Letter openings — **He** who holds/walks/is holy; **him** who overcomes |
| **5–6** | **Him** on throne; Lamb — **He** opens seals |
| **14** | With **Him** (Lamb); Son of Man on cloud — **His/He** |
| **17** | Lamb — **He** is Lord of lords; saints with **Him** |
| **19** | God’s judgments — **He/His**; rider — **He/Him/His** |
| **20** | **Him** on great white throne; beast’s **his** image |
| **21–22** | **He** wipes tears; **His** face/name; overcomer → **him/his** |
