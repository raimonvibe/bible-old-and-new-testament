# Pauline epistles capitalization pass (2 Cor – Philemon)

Applied 2026-07-22 via `scripts/apply-pauline-epistles-capitalization.js` (69 replacements total, including manual fixes for curly-quote mismatches).

## Books covered

2 Corinthians, Galatians, Ephesians, Philippians, Colossians, 1–2 Thessalonians, 1–2 Timothy, Titus, Philemon.

## Notable fixes

| Book / chapter | Fix |
|----------------|-----|
| **2 Cor 5** | **He** died for all → **Him** who rose; each judged by what **he** has done |
| **2 Cor 7** | Wronged party → **his** cause (human), not His |
| **2 Cor 10** | Boast in the Lord — **He** who boasts / let **him** boast |
| **Gal 3–5** | Abraham counted to **him**; persuasion from **Him** who calls; troubler → **he** |
| **Eph 1–2** | Full blessing/redemption passage — **He/Him/His** (God/Christ) |
| **Eph 4** | Psalm 68 citation — **He** ascended (Christ) |
| **Phil 2–3** | Kenosis hymn — **He** humbled Himself; know **Him** and His resurrection |
| **Col 1–2** | Christ hymn — **He/Him** throughout; **his** philosophy (human tradition) |
| **2 Thess 2** | Man of lawlessness — **he** exalts himself (not He) |
| **2 Tim 2** | Faithful saying — **Him/Himself** (Christ) |
