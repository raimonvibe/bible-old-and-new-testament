# 1 Corinthians capitalization pass

Applied 2026-07-22 via `scripts/apply-1-corinthians-capitalization.js` (35 replacements).

## Notable fixes

| Chapter | Fix |
|---------|-----|
| **1** | Enriched in **Him** (Christ); let **him** boast (human) |
| **2** | Spirit of man in **him**; natural man **he** can’t know |
| **3** | Paul/Apollos **he** who plants/waters; destroy **him** (temple defiler) |
| **6** | **he** joined to the Lord (believer) |
| **7** | Marriage/chastity — husbands and callings → **he/him** |
| **11** | Man’s **his** head (not He); Lord’s Supper **He** gave thanks; unworthy **he** eats |
| **14** | Tongues/prophecy — speakers are **he**, not He |
| **15** | Christ **He** was raised/appeared; **He** must reign |
| **16** | Timothy **he**; let **him** be cursed (apostate) |
